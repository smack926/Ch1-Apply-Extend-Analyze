Guideline 1.1: All visual elements must be accompanied by text descriptions, except in specific cases such as controls, media, tests, sensory experiences, CAPTCHA, or instances where the visuals serve only decorative purposes

Guideline 1.2: Prerecorded audio and video content should have additional elements like captions and descriptions, unless they're already being used as text. Live audio requires captions, and videos may need more detailed descriptions if they don't pause enough for clarity.
